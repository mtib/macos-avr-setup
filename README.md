# MacOS AVR Setup

Simple setup script to install and uninstall AVR tools.

## Usage

### One-liner

```
curl --proto '=https' --tlsv1.2 -sSf https://raw.githubusercontent.com/mtib/macos-avr-setup/main/install.sh | bash
```

### Manual

Download this repository and run `install.sh`.

## Included Programs

- avr-gcc
- avr-g++
- avrdude
- ... and many more
